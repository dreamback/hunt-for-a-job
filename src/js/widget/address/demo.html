联动地区选择

依赖

  1. address.css
  2. zepto.js
  3. address.js
  • 数据按需加载
  • 可以设置默认值
  • 可以验证是否已经合法选择
  • 三级/二级联动，或者只有省的一级选择

省市县/三级联动

html:

                
                
                
                //隐藏域添加默认值，省/市/县任意值

js:

                var ads = new Address({
                    province: $('#province'),
                    city: $('#city'),
                    country: $('#country'),
                    valueInput: $('#addressValue')//隐藏域ID
                });

验证是否已经有效选择：

                //使用check方法。
                $('#test').click(function(){
                    alert( ads.check() );
                });
